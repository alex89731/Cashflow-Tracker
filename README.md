# Веб-сервис для управления движением денежных средств (ДДС)

Это веб-приложение для учета, управления и анализа денежных операций.

## Установка и запуск

### 1. Клонирование репозитория

```bash
git clone <URL вашего репозитория>
cd <название папки проекта>
```

### 2. Создание и активация виртуального окружения

Для Windows:
```bash
python -m venv .venv
.venv\Scripts\activate
```

Для macOS/Linux:
```bash
python3 -m venv .venv
source .venv/bin/activate
```

### 3. Установка зависимостей

Установите все необходимые пакеты с помощью файла `requirements.txt`:

```bash
pip install -r requirements.txt
```

### 4. Настройка базы данных

Выполните миграции для создания таблиц в базе данных SQLite:

```bash
python manage.py makemigrations
python manage.py migrate
```

### 5. Создание суперадминистратора

Создайте учетную запись для доступа к админ-панели:

```bash
python manage.py createsuperuser
```
Следуйте инструкциям в консоли, чтобы задать имя пользователя и пароль.

### 6. Запуск веб-сервиса

Запустите сервер для разработки:

```bash
python manage.py runserver
```

После запуска сервис будет доступен по адресу [http://127.0.0.1:8000/](http://127.0.0.1:8000/).

## Использование

- **Главная страница** ([http://127.0.0.1:8000/](http://127.0.0.1:8000/)) — просмотр всех записей, фильтрация, создание, редактирование и удаление.
- **Админ-панель** ([http://127.0.0.1:8000/admin/](http://127.0.0.1:8000/admin/)) — управление справочниками (статусы, типы, категории, подкатегории).
